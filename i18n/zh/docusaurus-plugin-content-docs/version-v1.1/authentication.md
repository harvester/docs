在完成安装后的首次登录时，你会收到为默认 `admin` 用户设置密码的提示。

![auth](/img/v1.1/install/first-time-login.png)

:::note

在单集群模式下，只提供一个默认的 `admin` 用户。你可以查看 [Rancher 多集群](./rancher/rancher-integration.md)来进行多租户管理。

:::
